# Frontend mínimo — dentro de `frontend/`

Proyecto Next.js mínimo en TypeScript que sirve como punto de partida para la prueba "Excel Viewer & AI Modifier".

Cómo ejecutar (Windows PowerShell):

```powershell
cd frontend
npm install
npm run dev
```

La app estará disponible en http://localhost:3000

Notas:
- En próximos pasos implementaremos la UI para subir `.xlsx`, mostrar hojas y exportar la versión modificada por el backend.
